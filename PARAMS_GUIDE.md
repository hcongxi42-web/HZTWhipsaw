## 2. 筛选层参数

**位置：`batch_screen.py` 第 20-24 行**

```python
LOOKBACK_DAYS = 60    # 回溯交易日数
MIN_TURN = 2.0        # 日均换手率最低值 (%)
MIN_PRICE = 5.0       # 最低股价 (元)
MAX_PRICE = 500.0     # 最高股价 (元)
```

| 参数 | 默认值 | 含义 | 调大效果 | 调小效果 |
|------|--------|------|----------|----------|
| `LOOKBACK_DAYS` | 60 | 取多少天历史数据打分 | 需更长数据，次新股被淘汰更多 | 更多次新股进入候选 |
| `MIN_TURN` | 2.0 | 日均换手率门槛 | 只留活跃股，候选池缩小 | 纳入冷门股，候选池扩大 |
| `MIN_PRICE` | 5.0 | 最低股价 | 排除更多仙股 | 纳入低价股 |
| `MAX_PRICE` | 500.0 | 最高股价 | 纳入更高价股 | 排除高价股 |

**预筛选逻辑**（第 80-105 行 `quick_filter`）：

```python
# 振幅淘汰：近15日平均振幅 < 3% → 股性太死
if recent['amplitude'].mean() < 0.03:    # ← 改 0.03 调整振幅门槛

# 出货淘汰：近5日跌放量 >= 3天 → 疑似出货
if dist_days >= 3:                       # ← 改 3 调整容忍度

# 缩量回调必要：近15日无缩量下跌 → 无洗盘特征
retreat_mask = (recent['ret'] < 0) & (recent['vol_ratio'] < 0.85)  # ← 0.85 改缩量标准
```

## 3. 评分层参数

每维指标的**数学函数和可调参数**。

### 3.1 洗盘质量 `score_washout_quality`（权重 15%）

**度量什么**：下跌时是否缩量（洗盘特征 vs 出货特征）。

**子因子**：
| 子因子 | 权重 | 函数 | 关键参数 |
|--------|------|------|----------|
| 缩量程度 | 40% | sigmoid(avg_shrink, center=0.75, k=8) | `0.75` 中心点, `8` 陡峭度 |
| 缩量占比 | 35% | 线性: shrink_pct × 100 | `0.85` 缩量阈值 |
| 回撤深度 | 25% | bell(max_dd, mu=0.15, sigma=0.10) | `0.15` 最优回撤, `0.10` 宽度 |

**代码位置**：第 175-200 行

```python
# 关键参数
shrink_score = self._sigmoid(avg_shrink, 0.75, 8.0)   # 0.75 = 什么缩量程度算100分
shrink_pct = (down_days['vol_ratio'] < 0.85).sum()     # 0.85 = 量比多低算"缩量"
dd_score = self._bell(max_dd, 0.15, 0.10)              # 0.15 = 最优回撤深度
```

### 3.2 试盘信号 `score_probe_test`（权重 15%）

**度量什么**：是否有上影线+放量的试盘动作，以及试盘后的走势。

**子因子**：
| 子因子 | 权重 | 函数 | 关键参数 |
|--------|------|------|----------|
| 上影线质量 | 30% | 线性分段 [0.03→30, 0.12→90] | `0.03` 最低, `0.12` 最优 |
| 量能 | 25% | 线性分段 [1.2→30, 3.5→90] | `1.2` 最低, `3.5` 最优 |
| 试盘后走势 | 35% | bell(收益, 1.5%, 4%) + sigmoid(量, 0.85) | 收益中心 `0.015`, 量中心 `0.85` |
| 频率加成 | 10% | min(15, (n-1)×8) | `8` 每次加成 |

**代码位置**：第 202-248 行

```python
# 试盘识别阈值
probe_mask = (
    (d['vol_ratio'] > 1.2) &           # 量比门槛
    (d['upper_shadow_pct'] > 0.03) &   # 上影线最低长度
    (d['is_limit_up'] == 0) &          # 排除涨停
    (d['ret'] < 0.08)                  # 排除大涨
)
```

### 3.3 均线粘合 `score_ma_convergence`（权重 20%）

**度量什么**：MA5/10/20/60 四条均线的粘合程度，粘合越紧爆发力越强。

**子因子**：
| 子因子 | 权重 | 函数 | 关键参数 |
|--------|------|------|----------|
| 粘合度 | 55% | sigmoid(conv_ratio, 0.92, 20) | `0.92` 中心, `20` 陡峭 |
| 价格位置 | — | 线性: 12 - dev×180 | `12` 最高加分, `180` 衰减率 |
| 排列 | — | 多头+10/混乱0/空头-5 | — |
| 收敛加成 | — | sigmoid(delta, 0.003, 300) × 0.10 | `0.003` 敏感度 |

**代码位置**：第 307-357 行

```python
conv_score = self._sigmoid(conv_ratio, 0.92, 20.0)  # 0.92 = 什么粘合度算完美
pos_score = max(-5, 12 - price_dev * 180)            # 12 = 价格在均线中心时的最高分
```

### 3.4 股票强度 `score_stock_strength`（权重 30%，最大权重）

**度量什么**：洗盘前的上涨强度——好股票洗盘前应该有过像样的涨幅。

**子因子**：
| 子因子 | 权重 | 函数 | 关键参数 |
|--------|------|------|----------|
| 趋势强度 | 35% | bell(年化斜率, 25%, 18%) | `25` 最优年化涨幅, `18` 宽度 |
| 量能积累 | 25% | sigmoid(量比溢价, 1.08) + OBV | `1.08` 溢价中心 |
| 回调有序 | 25% | bell(回调深度, 0.25, 0.12) + sigmoid(缩量, 0.75) | `0.25` 最优回调 |
| 相对优势 | 15% | sigmoid(年超额, 8%) + sigmoid(胜率, 52%) | 需要 `index_daily` 表 |

**代码位置**：第 421-594 行

```python
# 核心参数
slope_score = self._bell(annual_slope, 25, 18)        # 25% = 最优年化趋势
depth_score = self._bell(dd, 0.25, 0.12)              # 25% = 最优回调深度
excess_score = self._sigmoid(annual_excess, 8, 0.06)   # 8% = 年超额收益中心
```

### 3.5 启动准备 `score_launch_readiness`（权重 10%）

**度量什么**：最近是否有质量阳线（放量上涨非涨停）+ 均线配合。

**子因子**：
| 子因子 | 权重 | 函数 | 关键参数 |
|--------|------|------|----------|
| 质量阳线 | 55% | bell(涨幅, 4.5%, 2.5%) + bell(量比, 1.5) | `4.5%` 最优涨幅, `1.5` 最优量比 |
| 均线配合 | 15% | 站上N条均线: -8/0/6/12 | — |
| 稳定性 | 10% | sigmoid(振幅std, -200) + bell(平坦度) | — |
| 新鲜度 | 20% | days_ago × 5, max 20 衰减 | `5` 每日衰减 |

**代码位置**：第 250-305 行

```python
quality_up = d[(d['ret'] > 0.02) & (d['vol_ratio'] > 1.0) & (d['is_limit_up'] == 0)]
# 0.02 = 质量阳线最低涨幅
# 1.0  = 质量阳线最低量比
```

### 3.6 量价健康 `score_volume_price_health`（权重 10%）

**度量什么**：量价配合是否健康（涨放量跌缩量 = 好）。

**子因子**（由 `fund_flow` + `volume_health` 等权合并）：
| 子因子 | 权重 | 函数 | 关键参数 |
|--------|------|------|----------|
| 健康日占比 | 35% | sigmoid(%, 0.25, 10) | `0.25` 健康比例中心 |
| 出货惩罚 | 15% | 100 - sigmoid(%, 0.15, 15) | `0.15` 出货比例容忍度 |
| 洗盘加成 | 30% | sigmoid(%, 0.20, 8) | `0.20` 缩量下跌比例 |
| 量价同步 | 20% | (sync-0.50)×200 | — |

## 4. 权重配置

**位置：`batch_screen.py` 第 608-615 行 `compute_total_score()`**

```python
weights = {
    'washout_quality':      0.15,   # 洗盘质量
    'probe_test':           0.15,   # 试盘信号
    'ma_convergence':       0.20,   # 均线粘合
    'stock_strength':       0.30,   # 股票强度 ← 最大权重
    'launch_readiness':     0.10,   # 启动准备
    'volume_price_health':  0.10,   # 量价健康
}
# 总和必须 = 1.0
```

**调权指南**：

| 你想强调... | 调高 | 对应降低 |
|-------------|------|----------|
| 股票自身趋势质量 | `stock_strength` | `ma_convergence` |
| 洗盘形态纯度 | `washout_quality` | `stock_strength` |
| 爆发时机 | `launch_readiness` | `probe_test` |
| 试盘信号灵敏度 | `probe_test` | `volume_price_health` |

## 5. 修改实操

### 5.1 完整流程（改一个参数 → 前端生效）

```
① 编辑 batch_screen.py 改参数
       │
② 清空旧评分（可选，推荐）
       │  python -c "import sqlite3; c=sqlite3.connect('stock_data.db');
       │              c.execute('DELETE FROM screening_history'); c.commit()"
       │
③ 重跑评分
       │  python batch_screen.py --latest
       │  (或 python batch_screen.py --date 2026-06-24 只跑一天测试)
       │
④ 导出静态站点
       │  python generate_static.py
       │
⑤ 本地验证（可选）
       │  浏览器打开 docs/index.html 查看排名变化
       │
⑥ 部署
       │  git add docs/ && git commit -m "调整权重/参数" && git push
```

### 5.2 快速测试（只跑一天看效果）

```bash
# 不删旧数据，只重跑最近一天，快速对比
python batch_screen.py --date 2026-06-24
python generate_static.py
# 打开 docs/index.html → 选 06-24 → 对比排名变化
```

### 5.3 参数回滚

```bash
git checkout -- batch_screen.py      # 恢复到上次提交的版本
python batch_screen.py --latest      # 用旧参数重跑
python generate_static.py
```

## 6. 参数速查表

### 连续函数工具箱

| 函数 | 公式 | 用途 |
|------|------|------|
| `_sigmoid(x, center, k)` | 100 / (1+e^(-k(x-center))) | "越大越好"或"越小越好"(k取负) |
| `_bell(x, mu, sigma)` | 100 × e^(-((x-mu)/σ)²) | "最优区间"，偏离即衰减 |

### 所有可调参数一览

| 参数 | 位置(行) | 默认值 | 调大 | 调小 |
|------|----------|--------|------|------|
| `LOOKBACK_DAYS` | 20 | 60 | 需更长历史 | 纳入次新股 |
| `MIN_TURN` | 21 | 2.0 | 更活跃 | 更宽松 |
| `MIN_PRICE` | 22 | 5.0 | 排除更多低价 | 纳入更多 |
| `MAX_PRICE` | 23 | 500 | 纳入更高价 | 排除高价 |
| 缩量中心 | 186 | 0.75 | 更严格 | 更宽松 |
| 回撤最优 | 195 | 0.15 | 接受更深回调 | 偏好浅回调 |
| 试盘量比门槛 | 206 | 1.2 | 更严格 | 更敏感 |
| 试盘上影线门槛 | 207 | 0.03 | 需更长上影线 | 更敏感 |
| 粘合度中心 | 325 | 0.92 | 更严格 | 更宽松 |
| 趋势最优年化 | 463 | 25 | 偏好更强趋势 | 接受温和趋势 |
| 回调最优深度 | 527 | 0.25 | 偏好深回调 | 偏好浅回调 |
| 质量阳线最低涨幅 | 256 | 0.02 | 需更大涨幅 | 接受小涨 |
| **6维权重** | **608-615** | — | — | **总和=1.0** |
