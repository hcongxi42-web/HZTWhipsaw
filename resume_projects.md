2026.06–至今  量价洗盘评分系统 HZTwhipsaw（Vibe coding）

· 项目概况：构建全市场 A 股量价洗盘自动评分与可视化平台。每日自动拉取行情数据，从洗盘质量、试盘信号、均线粘合、启动准备、资金流向、量价健康六个维度对全市场股票打分排名，辅助识别主力洗盘后即将启动的标的。
· 项目内容：
  - 设计六维加权评分模型（洗盘质量 23%、试盘信号 23%、均线粘合 17%、启动准备 13%、资金流向 12%、量价健康 12%），覆盖 33 个量价指标，含试盘检测、均线粘合度、量价四象限分析等自研因子。
  - 搭建 GitHub Actions 自动化数据流水线，每日北京时间 06:00 自动拉取 baostock 行情 → 筛选打分 → 生成静态站点 → 部署 GitHub Pages，失败 09:00 自动重试。
  - 纯前端交互式仪表板：支持按日期/行业/板块多维筛选、个股六维雷达图、历史得分排名走势、K 线图，采用学术期刊风格设计（Times New Roman、低饱和配色、半去框化）。
· 技术栈：Python（Pandas/NumPy/SciPy）+ baostock + GitHub Actions + GitHub Pages + ECharts
· 查看页面：https://hcongxi42-web.github.io/HZTWhipsaw/
